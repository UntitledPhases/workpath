# Visual Grammar

- Step: rectangle
- Gate: diamond
- Evidence: ellipse
- Boundary / handoff: hexagon

V1 step modules:

- intent
- research
- plan
- execute
- verify
- return

V1 edge kinds:

- produces
- validates
- gates
- hands_off_to

The canvas should show compact titles and shape language. Full metadata belongs
in the side panel.
